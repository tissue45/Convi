## 🏗️ 시스템 구성

### Frontend Architecture
```
React 19 App
├── Customer App        # 고객용 인터페이스
├── Store Owner App     # 점주용 관리 시스템
├── Headquarters App    # 본사용 관리 시스템
└── Common Components   # 공통 컴포넌트
```

### Backend Services
```
Supabase Backend
├── PostgreSQL Database    # 메인 데이터베이스
├── Authentication         # 사용자 인증
├── Real-time Subscriptions # 실시간 데이터
├── Storage                # 파일 저장소
└── Edge Functions         # 서버리스 함수
```

### External Integrations
- **토스페이먼츠**: 결제 처리
- **Naver Maps API**: 지도 및 위치 서비스
- **Render.com**: 배포 및 호스팅
